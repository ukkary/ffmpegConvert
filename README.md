# ffmpegConvert.py

## about
inputファイルの中のmp4を一括変換してmp4に変換してoutputに書き出す（笑）なコード   　
拡張子やエンコード方式をかえれば少しは実用的になるかも？

## install
ffmpeg and terminal notifier   
``` brew install ffmpeg ```   
and   
``` brew install terminal-notifier ```

## usage
inputファイルの中にフォルダもしくはファイルを配置後   
```python ffmpegConvert.py```
